Anatomy of a Prompt

  1. Role: "You are a creative coder who generates single-page HTML artifacts."
  2. Constraint: "Use only vanilla JavaScript — no external libraries."
  3. Aesthetic: "Monochrome, geometric, generative — early computer art."
  4. Technical: "Canvas API, full viewport, requestAnimationFrame."
  5. Output: "Return a single HTML file. No explanation — just code."
